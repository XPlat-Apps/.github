# XPlat Apps

> [!WARNING]
> This GitHub organization is archived and no longer maintained.

XPlat Apps was an open-source project focused on bringing UWP-style APIs to cross-platform .NET applications during the Xamarin/UWP era.

The repositories in this organization are preserved for historical and reference purposes only. They should not be used for new projects.

If you've relied on these packages over the years, thank you very much for leveraging them for your applications. 

All related NuGet packages have been deprecated. Older package versions have also been unlisted where appropriate.

For modern cross-platform .NET development, consider current .NET, .NET MAUI, or native platform APIs.

## Status

- Repositories: archived
- NuGet packages: deprecated
- Maintenance: ended
- Support: no longer provided
